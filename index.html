About

Hydrogen is an advanced, so called "ghost client" for Minecraft version 1.8.9, based on Minecraft Forge. Hydrogen is made for the pupose of giving yourself various utilites, which all give you an advantage without anyone noticing it, including many render related modules.

straight-forward

Hydrogen is straight to the point. Each module has a description of what it does and are generally easy to use and understand.

open-source

Hydrogen is completely open-source, hosted on our GitHub. No sneaky bitcoin-miners and IP-Loggers. If you want to contribute to Hydrogen in any way, by suggestions or similar, feel free to do so!

easy-to-use advanced gui

Hydrogen's GUI is easy to use, understand and highly customizable. Want to use a different font instead of the normal Minecraft font? You're free to do so! Dont like the Arraylist or Watermark? Disable them! You have tons of options!

Download


Hydrogen is available to download at either on this site or on our GitHub. If you want an automatic install, download our Installer down below or download the jar in itself for a manual installation.

To install Hydrogen automatically, open the Installer, install the Version you want and then start a version of Minecraft Forge 1.8.9.

Dont have Forge installed? Download it here!

To install Hydrogen manually, put the jar you download inside /.minecraft/mods, and then run a version of Minecraft Forge 1.8.9.
